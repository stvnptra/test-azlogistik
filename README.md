Untuk initiate lakukan instalasi package
```bash
npm i
```
gunakan ``` --legacy-peer-deps``` ketika ada kendala saat instalasi

setelah itu dapat melakukakan serve menggunakan 
```bash
npm run dev
```

Techstack 
```
NextJs
Tailwind
Typescript
```
